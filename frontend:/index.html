AgriSense AI runs on a distributed microservice architecture. All microservices communicate locally via REST APIs. Review the diagnostics matrix below to verify active liveness probes.

Predict Service Port 8000

Executes predictions using a Random Forest classifier. Reads pH, temperature, humidity, and rainfall.

offline /predict
Profit Calculator Port 8001

Fetches wholesale crop prices from Agmarknet (2024-25 modal values) and calculates production costs.

offline /profit
Weather & Geocoding Port 8002

Pings Open-Meteo APIs to geocode location strings, fetch 14-day rainfall forecasts, and extract climate data.

offline /weather/geocode
Risk Assessment Engine Port 8003

Processes climate variables against optimal thresholds to calculate crop soil, water, weather, and disease risk ratings.

offline /risk
AI Gemini Advisor Port 8004

Interfaces with Gemini 1.5 to provide natural language agronomic feedback, cost assessments, and crop schedules.

offline /advise
Crop operations Planner Port 8005

Matches crops against optimal planting windows, reading crop lifecycle schedules from JSON metadata.

offline /planner/schedule
📡 Live Sensor Feed Port 8006

IoT sensor abstraction layer. Streams field readings (temperature, humidity, soil moisture, soil pH) to the dashboard in real time.

offline /latest-sensor
